# Scene Saver Extension

[Website/Releases](http://www.polydes.com/repo/toolset/com.polydes.scenesaver) |
[Issues](https://github.com/polydes/scenesaver/issues)

</div>

This [Stencyl](https://www.stencyl.com/) extension provides the capability to send scene data from a running Stencyl game back into the toolset. This can be handy for games that include a custom level editor, allowing scenes to be edited in-game and then the result saved back to the editor.

## Using this Stencyl extension

This is a free extension. Installation instructions and guides can be found [here](http://www.polydes.com/repo/toolset/com.polydes.scenesaver/).
